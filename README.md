# Консольная утилита для анализа данных о подготовке студентов к экзаменам.
Скрипт читает файлы в формате csv с данными о подготовке студентов к экзаменам и формирует отчеты. Название файлов (может быть несколько) и название отчета передается в виде параметров --files и --report. Отчёт формируется по всем переданным файлам, а не по каждому отдельно. Отчет в виде таблицы выводится в консоль.

## Добавление новых отчетов

1) Создайте класс, унаследованный от Report в пакете reports

2) Реализуйте методы name, calculate, headers

3) Зарегистрируйте в фабрике в __init__.py пакета reports

## Пример запуска
`python3 main.py --files data/math.csv data/physics.csv data/programming.csv --report median-coffee`
